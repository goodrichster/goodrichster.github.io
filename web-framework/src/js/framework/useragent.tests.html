useragent

Adds classes to the html tag describing the user agent

  • html.ua-no-touch - not a touch device
  • html.ua-touch - a touch device
  • html.ua-desktop - a device within the desktop breakpoint
  • html.ua-tablet - a device within the tablet breakpoint
  • html.ua-mobile - a device within the mobile breakpoint
  • html.ua-js .ua-js-visible - ondomready visibility
