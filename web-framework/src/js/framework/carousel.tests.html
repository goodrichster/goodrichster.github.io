carousel

The carousel module activates page carousels.

Use the .has-shuffle class on the .carousel-container to randomize the first item.

Use the .has-memory class combined with an id on the .carousel-container to provide memory to the carousel.

Use the .carousel-nav-minimal on the .carousel-nav remove the arrows.

Use the .carousel-nav-centered on the .carousel-nav to center the navigation.

Use the .carousel-panels-fixed-width on the .carousel-panels to force the size of the panels.
